# Profit_analysis_project
Built a multiple linear regression model to analyze the impact of R&amp;D, marketing, and administration spending on startup profit and predicted future profit scenarios using Python and data visualization tools.
# Profit Analysis Using Multiple Linear Regression

## 📌 Project Overview
This project analyzes startup financial data to understand how R&D, Marketing, and Administration spending affect company profit. A multiple linear regression model is built to predict profit for new investment scenarios.

## 📊 Dataset
- Dataset: 50 Startups
- Records: 50
- Features:
  - R&D Spend
  - Administration
  - Marketing Spend
  - State
  - Profit

## 🛠 Tools & Technologies
- Python
- Pandas, NumPy
- Scikit-learn
- Matplotlib, Seaborn
- Power BI / Tableau

## 🔍 Methodology
1. Data loading and cleaning
2. One-hot encoding for categorical variables
3. Train-test split
4. Multiple Linear Regression model
5. Model evaluation using R² score
6. Profit prediction
7. Data visualization and insights

## 📈 Key Insights
- R&D spending has the highest impact on profit
- Marketing spend has moderate impact
- Administration spend has minimal influence
- Optimizing R&D investment improves profitability

## 📌 Conclusion
The project demonstrates how regression models and data visualization can support data-driven business decisions.
